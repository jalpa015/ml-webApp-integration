ML Recommender System
This Web App is a Machine Learning Recommender System that recommends movies which are similar to the movie entered in the search box below.
The Machine Learning model has been trained on a dataset of approximately 6000 movies. The input features of the ML model consisted of the cast, genre, title and plot of the movie.
